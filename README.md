# 📊 Customer Shopping Behavior Analysis

## 📌 Overview
This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI.  
The goal is to extract meaningful business insights from raw data through data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard visualization.

The project demonstrates end-to-end data analytics workflow — from data loading to reporting.

---

## 📂 Dataset
The dataset contains customer purchase transactions, including:
- Customer details
- Product information
- Order dates
- Quantity purchased
- Sales amount
- Category information

The data was cleaned and prepared before analysis to ensure accuracy and reliability.

---

## 🛠 Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL** (PostgreSQL / MySQL / SQL Server)
- **Power BI**
- **Jupyter Notebook**

---

## 🔎 Project Steps

### 1️⃣ Data Loading
- Imported dataset into Python using Pandas
- Performed initial data inspection

### 2️⃣ Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected data types
- Standardized column names

### 3️⃣ Exploratory Data Analysis (EDA)
- Analyzed sales trends
- Identified top-selling products
- Examined customer purchasing behavior
- Visualized key patterns using charts

### 4️⃣ SQL Analysis
- Imported cleaned data into database
- Wrote SQL queries to:
  - Calculate total revenue
  - Identify top customers
  - Analyze category performance
  - Track monthly sales trends

### 5️⃣ Dashboard Creation
- Built an interactive Power BI dashboard
- Added KPIs, slicers, and visualizations
- Designed for business decision-making

### 6️⃣ Reporting
- Created a structured report summarizing:
  - Key insights
  - Business recommendations
  - Data-driven conclusions
